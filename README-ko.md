# AIOpsPilot

자율 클라우드 운영 컨트롤 플레인 — **AIOps** 접근이며 초기 버티컬은
**Resilience**(재해 복구·chaos/리질리언스 테스트), **Change Safety**(안전한 변경·드리프트
조치), **Cost Governance**(FinOps). 같은 아키텍처가 다른 AIOps 도메인(posture management,
SRE/SLO 등)에도 적용되며 이는 향후 범위입니다. 대부분의 이벤트를 결정론적으로 해결하고 LLM은
잔여 모호한 경우에만 사용해 사람 개입을 최소화합니다.
