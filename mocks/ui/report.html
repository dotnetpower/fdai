AS-IS  →  TO-BE

Where a general-purpose agent spends effort vs the deterministic-first design.

AS-IS · general agent

  • Every event re-runs an LLM from scratch (diagnose, RCA, remediate).
  • Frequent escalation; humans confirm most actions.
  • Latency bound to model round-trips.
  • Cost scales with token volume.

TO-BE · deterministic-first

  • T0/T1 resolve the large majority without a frontier model.
  • Risk-gated autonomy; humans see only high-risk and novel cases.
  • T0/T1 answer in milliseconds to seconds.
  • Only a small share reaches a frontier model.
