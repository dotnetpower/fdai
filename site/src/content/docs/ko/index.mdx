## 자율성이 여러분에게 주는 것

<ul class="impact-list">
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <strong>이벤트의 ~70-80%가 결정론적으로 해소.</strong> 규칙 · 정책 ·
    체크리스트가 반복 가능한 다수를 결정. 모델 호출 없음.
  </li>
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <strong>~15-20%는 경량 패턴 재사용(T1)으로 해소.</strong> 과거 해소된
    인시던트와의 임베딩 유사도, 소형 분류기, frontier 모델 없음.
  </li>
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <strong>추론 티어(T2)에 도달하는 것은 ~5-10%뿐.</strong> mixed-model
    교차 검증과 결정론적 검증기 - 모델이 제안하면 검증기가 최종 판정합니다.
  </li>
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <strong>모든 자율 액션은 네 가지 불변식을 지닙니다.</strong> stop-condition ·
    롤백 경로 · 영향 범위 제한 · 실행 기록 항목. 넷 중 하나라도 없으면
    액션은 자동 실행되지 않고 사람 승인을 요청합니다.
  </li>
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <strong>새 능력은 관찰 모드로 시작하며, 예고 없이 등장하지 않습니다.</strong>
    변경 적용은 Phase 0 기준선 대비 별도의 측정 가능한 검토 단계입니다.
  </li>
  <li>
    <span class="impact-tick" aria-hidden="true">↳</span>
    <em>위 수치는 모두 설계 목표이지 측정된 결과가 아닙니다.</em>
    Forward Deployed Agents는 짝을 이룬 기준선 측정 없이 배수를 주장하지 않습니다.
  </li>
</ul>

## 에이전트 조직을 만나보세요

<p class="hero-note">15개의 이름을 가진 에이전트가 하나의 조직으로 제어 루프를 운영합니다 - 각자 역할을 맡고 스키마 검증된 이벤트 버스로 인계합니다. 은유가 아니라 조직도입니다: 판단자는 실행하지 않으며, 모든 액션은 서로 다른 에이전트가 개시, 판단, 승인, 실행, 감사합니다.</p>

<CardGrid>
  <Card title="Odin - 지휘" icon="star">
    마스터 플래너. 버티컬 간 중재와 판정 확정 전 최종 타이브레이크.
  </Card>
  <Card title="판단과 실행" icon="approve-check">
    **Forseti**가 판단(자동 실행 / 사람 승인 / 차단), **Thor**가 디스패치·실행, **Var**가 사람 승인, **Vidar**가 롤백·DR 페일오버를 담당합니다. 어떤 에이전트도 판단과 실행을 겸하지 않습니다.
  </Card>
  <Card title="감지" icon="magnifier">
    **Huginn**이 이벤트를 수집·상관하고, **Heimdall**이 상태와 드리프트를 관측합니다. 결정론 우선 - 핫패스에 LLM은 없습니다.
  </Card>
  <Card title="대화와 감사" icon="comment">
    **Bragi**가 운영자 질문을 도구 호출로 번역하고, **Saga**가 append-only 감사 로그를 쓰고 인계를 에스컬레이션합니다.
  </Card>
  <Card title="거버넌스" icon="setting">
    **Mimir**가 룰 카탈로그를 관리하고, **Norns**가 결과에서 학습하며, **Muninn**이 장기 기억을 보유합니다.
  </Card>
  <Card title="도메인 전문가" icon="puzzle">
    **Njord**(비용), **Freyr**(용량), **Loki**(카오스)가 도메인 분석으로 판단자를 보조합니다 - 직접 실행하지 않습니다.
  </Card>
</CardGrid>
